OSINT METHODOLOGY:
1. Start with domain intelligence:
   - osint_recon action=whois → registration details
   - osint_recon action=dns → DNS records
   - osint_recon action=harvest → emails, hosts, employees
2. Generate and execute Google dorks
3. For discovered emails:
   - social_profile search across platforms
   - darkweb_search for breach data
   - Check GitHub for commits by target employees
4. For phone numbers (if provided):
   - phone_lookup sources=all → carrier, location, social
   - Chain to social profiles and breach data
5. For discovered usernames:
   - social_profile tool=maigret (deep enumeration)
6. Check public code repositories:
   - GitHub/GitLab for leaked secrets, API keys, credentials
   - Search for target domain in repos

username → 500+ platform check → profile data → personal info
domain → WHOIS → emails → employees → social engineering targets
GitHub repo → commit history → email addresses → API keys/secrets
