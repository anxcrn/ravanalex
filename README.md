## рҹҡҖ HOW TO RUN

</div>

### Prerequisites
- [Bun](https://bun.sh) v1.2+
- Node.js 20+
- Git
- An LLM API key (Anthropic/OpenAI/Google Gemini)

---

### рҹ–ҘпёҸ Option 1 вҖ” Run Locally (Windows/macOS/Linux)

```bash
# 1. Clone the repo
git clone https://github.com/anxcrn/ravanalex.git
cd ravanalex

# 2. Install dependencies
bun install

# 3. Set your API key (choose one)
export ANTHROPIC_API_KEY=sk-ant-...
export OPENAI_API_KEY=sk-...
export GEMINI_API_KEY=AIza...

# 4. Run ALEX
bun run dev

# OR build and run the CLI directly
bun run build
./packages/alex/bin/alex
```

---

### вҳҒпёҸ Option 2 вҖ” Deploy on Oracle Cloud FREE VPS (Recommended)

Oracle Cloud gives you **4 ARM cores + 24GB RAM + 200GB SSD вҖ” always free, forever.** This is the correct way to run ALEX вҖ” always on, always hunting.

#### Step 1: Get Your Free Oracle VPS
1. Go to **[cloud.oracle.com](https://cloud.oracle.com)** вҶ’ Create Free Account
2. **Compute вҶ’ Instances вҶ’ Create Instance**
3. Settings:
   - **Image**: Ubuntu 22.04 LTS
   - **Shape**: Ampere A1.Flex вҖ” **4 OCPUs, 24GB RAM** (Always Free)
   - **Boot Volume**: 200 GB
   - **SSH Key**: Upload or download generated key

#### Step 2: SSH Into Your VPS
```bash
ssh -i your-key.pem ubuntu@YOUR_ORACLE_IP
```

#### Step 3: One-Command ALEX Setup
```bash
# Paste this on your Oracle VPS вҖ” does EVERYTHING automatically
bash <(curl -fsSL https://raw.githubusercontent.com/anxcrn/ravanalex/main/deploy/setup.sh)
```

**What the script does:**
- Updates Ubuntu packages
- Installs Bun, Node.js, Git, nmap, hashcat, metasploit, and all required tools
- Clones RAVAN-ALEX from GitHub
- Configures systemd service for 24/7 uptime (auto-restarts on crash)
- Sets up firewall rules
- Prompts for your LLM API key
- Starts ALEX

#### Step 4: Open in Browser
```
http://YOUR_ORACLE_IP:3000
```

ALEX is now running 24/7 on your free Oracle VPS. рҹ”Ҙ
